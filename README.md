# Task 1: Linear Regression 📈

This project was completed as a part of the **Data Science Internship** at **Prodigy Infotech**.

## 📌 Objective

To implement a **Linear Regression Model** to predict dependent variables based on independent variables using Python and visualize the results.

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📂 Files

- `task1_linear.zip`: Contains the Jupyter Notebook and dataset used for training the linear regression model.
- `.gitattributes`: Git LFS tracking file

## 🧠 Key Learnings

- Understanding Linear Regression from scratch
- Data preprocessing and handling missing values
- Model fitting using scikit-learn
- Evaluation using visualizations
